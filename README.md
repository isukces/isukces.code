# iSukces.Code

Niewielki framework do generowania kodu C# i TypeScript. 
Pozwala budować pliki dobrze sformatowane i czytelne pliki `cs` i `ts`. 
Sprawdza się przy tworzeniu powtarzalnego kodu jak metody Equals, Clone itp.
W repo znajdują się m.in. biblioteki `iSukces.Code` (fluent API do tworzenia kodu), 
`iSukces.Code.AutoCode` (generatory oparte o refleksję/atrybuty).
